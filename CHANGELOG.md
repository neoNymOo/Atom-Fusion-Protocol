# Changelog

## [2.0.0] - 2025-29-12

### Added
- Добавлена настройка хитбокса сущности, которая реализует поломанный экзоскелет.
- Добавлена настройка вероятности появления поломанных элементов брони.
- Добавлена настройка вероятности для появления разных типов брони.
- Добавлена настройка вероятности для появления разных частей брони.
- Изменена модель экзоскелета.
- Экзоскелету добавлен иммунитет к огню.
- Добавлена поломанная силовая броня и ее генерация в мире.
- Значительно улучшена производительность рендера моделей.
- Теперь корректно отображается броня на руке игрока.
- Улучшено качество моделей брони.
